# 🎵 Spotify Real-Time Dashboard

A dynamic, interactive Power BI dashboard designed to explore real-time Spotify music trends, track performance, and user engagement patterns. This dashboard provides actionable insights for music analysts, artists, and data enthusiasts.

---

## Short Description / Purpose
The Spotify Real-Time Dashboard visualizes authentic streaming data to help users analyze top tracks, artists, and genres in real time. It enables data-driven decisions and insights for understanding global listening patterns and music popularity trends.

## Tech Stack
The dashboard was built using the following tools and technologies:  
- 📊 **Power BI Desktop** – Primary tool for report creation and data visualization.  
- 📂 **Power Query** – Data transformation and preparation.  
- 🧠 **DAX (Data Analysis Expressions)** – Calculated measures, dynamic visuals, and conditional logic.  
- 📝 **Data Modeling** – Relationships created among tables to enable cross-filtering and aggregation.  
- 📁 **File Format** – `.pbix` for the working dashboard.

---

## Data Source
- **Source:** Maven Analytics  
- **Description:** Real-life Spotify streaming data including track performance, artist popularity, genres, and regional metrics.  
- Structured to allow analysis of top-performing songs, artists, and trends in listening behavior globally.

## Features / Highlights

### Business Problem
The music industry generates billions in streaming revenue, yet understanding real-time trends across tracks, artists, and regions can be complex. Questions such as:  
- Which tracks are trending right now globally?  
- Which artists dominate certain regions or genres?  
- How does listener engagement vary by time or country?  
… are difficult to answer quickly with raw data.

### Goal of the Dashboard
To deliver an interactive, real-time visualization tool that:  
- Enables exploration of Spotify streaming trends.  
- Supports decision-making for artists, music producers, and analysts.  
- Identifies patterns in listener behavior, top tracks, and genre performance.

### Walkthrough of Key Visuals
- **Top KPIs:**  
  - Current top tracks, top artists, and trending genres.  
  - Total streams analyzed and regional engagement metrics.  
- **Track & Artist Filter Panel:**  
  Interactive slicers for filtering visuals by artist, track, or genre.  
- **Top Tracks & Artists (Bar Charts):**  
  Shows leading tracks and artists globally or by region.  
- **Genre Distribution (Pie/Donut Charts):**  
  Highlights the popularity of different genres among listeners.  
- **Streaming Trends (Line Charts):**  
  Tracks performance over time to spot spikes in streams or listener activity.  
- **Regional Insights (Map Visuals):**  
  Displays global listening patterns and top countries for specific tracks/artists.

### Business Impact & Insights
- **Marketing Optimization:** Identify which tracks or artists are trending in specific regions for promotional campaigns.  
- **Strategic Releases:** Artists and labels can plan releases based on listening trends and engagement patterns.  
- **Audience Analysis:** Understand listener preferences by region, genre, or time to optimize playlists.  
- **Music Analytics:** Provides a clear, interactive view of real-time streaming data for analysts and decision-makers.

## How to Use
1. Open the `.pbix` file in **Power BI Desktop** or **Power BI Service**.  
2. Apply filters and slicers to navigate through tracks, artists, and genres.  
3. Use **full-screen mode** for presentations.
