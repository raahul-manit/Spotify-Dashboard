# Spotify Dashboard

A **real-time Spotify dashboard** built in Power BI to explore music trends and user engagement.

## Data Source
- Real-life data from **Maven Analytics**  
- Reflects authentic streaming patterns and user behavior

## Key Insights
- Top tracks and artists in real-time  
- Streaming trends across regions  
- Popular genres and listening patterns  
- Comparative performance of playlists and songs

## Features
- Interactive visuals for tracks, artists, and genres  
- Dynamic filters for detailed exploration  
- Full-screen view for presentations

## How to Use
1. Open the `.pbix` file in **Power BI Desktop** or **Power BI Service**  
2. Apply filters or slicers to navigate through data  
3. Use full-screen mode for presentations **
